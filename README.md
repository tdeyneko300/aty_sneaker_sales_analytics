# aty_sneaker_sales_analytics
1. Краткое описание проекта

Цель: Аналитика продаж онлайн-магазина кроссовок с китайского маркетплейса Poizon. Миграция Excel → SQLite БД → построение дашборда

Задача: Из плоского Excel файла → нормализованная БД → бизнес-метрики (топ-бренды, сегментация клиентов, контроль партий).

2. Стек и требования
БД: SQLite 3 (aty_database 1.db)

Дашборд: Yandex DataLens https://datalens.yandex/qgn1yxm05yiga

Инструмент: DB Browser for SQLite, Yandex DataLens

Данные: Реальный Excel от магазина
